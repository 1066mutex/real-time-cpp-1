Companion Code for Real-Time C++.

The reference project supports ATMEL(R) AVR(R) Atmega328P,
Win32 (Release + Debug), and ST microelectronics(R)
STM32F10x ARM(R) Cortex(TM)-M3.

There is workspace support for
Microsoft(R) Visual Studio(R) 2012 and
ATMEL Studio(R) 6.
